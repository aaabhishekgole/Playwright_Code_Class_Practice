# Playwright Code Class Practice ✅

This repository contains practice exercises and example scripts for Playwright with JavaScript. It's organized into chapters and example files to help you learn and run simple Playwright tests and Node.js scripts.

## Repository structure

- Chapter01_Basics/
  - `01_basic.js` — simple Node.js example
  - `02_.js`
  - `03_verifySetup.js` — verify setup examples
  - `04_hotCode.js`

## Quick start ⚡

Prerequisites:
- Node.js (>= 16)

Install dependencies:
```bash
npm install
```

Install Playwright browsers (if you plan to run tests):
```bash
npx playwright install
```

Run an example script:
```bash
node Chapter01_Basics/01_basic.js
```

Run Playwright tests:
```bash
npx playwright test
```
